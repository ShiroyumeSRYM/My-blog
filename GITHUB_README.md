# シロユメ_Shiroyume 个人博客

> 一个现代化的个人博客网站，采用 Windows 10 风格的毛玻璃 UI 设计，支持多语言和响应式布局。

![个人博客](https://img.shields.io/badge/平台-GitHub%20Pages-blue?style=flat-square)
![语言-JavaScript-yellow?style=flat-square)
![框架-原生%20HTML5-green?style=flat-square)
![样式-CSS3-blue?style=flat-square)

## ✨ 特性

- 🎨 **毛玻璃效果UI** - 现代化的半透明设计
- 🌍 **多语言支持** - 中文、日语、英语自动切换
- 📍 **智能语言检测** - 基于IP地址的自动语言设置
- 📱 **完全响应式** - 完美适配手机、平板、桌面
- 🎵 **作品展示** - 集成B站、YouTube、网易云等平台
- 💬 **动态弹窗** - 优雅的作品列表展示
- ⚡ **零依赖部署** - 纯静态文件，无需后端

## 🚀 快速开始

### 在线体验
👉 **立即访问**: [在线演示地址](https://[你的用户名].github.io/[仓库名]/)

### 本地运行
1. 克隆仓库：`git clone [仓库地址]`
2. 直接打开 `index.html`
3. 开始使用！

### 添加新作品
编辑 `popup.json` 文件即可添加新作品，支持热更新！

## 🎯 部署

### 一键部署（推荐）

```bash
# Windows 用户
.\deploy.bat

# Mac/Linux 用户  
./deploy.sh
```

### 手动部署

1. 点击上方 "Use this template" 按钮
2. 创建新仓库
3. 上传所有文件
4. 在 Settings → Pages 中启用

## 🎨 自定义

### 🖼️ 头像和背景
- 头像：替换 `assets/images/avatar.jpg`
- 背景：在 CSS 中取消注释背景图片

### 🔗 社交链接
编辑 `index.html` 中的社交媒体链接

### 🎵 作品平台
在 `popup.json` 中添加新作品和平台链接
